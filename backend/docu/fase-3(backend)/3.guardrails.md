def verificar_mensagem(mensagem: str) -> dict:
    prompt = f"""
Analise se essa mensagem é segura para um assistente financeiro responder.
Retorne APENAS um JSON:

{{
    "permitido": true ou false,
    "motivo": "topico_bloqueado" | "dado_sensivel" | "valor_invalido" | null,
    "resposta": "mensagem ao usuário se bloqueado, null se permitido"
}}

Bloquear se:
- Pedir ajuda com fraude, corrupção ou atividades ilegais
- Conter dados pessoais sensíveis (CPF, senha, conta bancária)
- Não tiver nada a ver com concurso público ou custo de vida em SP

Mensagem: "{mensagem}"
"""
```

✅ Mais inteligente — entende contexto
✅ Sem lista de palavras pra manter
❌ Custa uma chamada extra à API
❌ Mais lento
```
