# É o Django REST Framework (DRF) — uma biblioteca que transforma seu Django numa API REST de forma fácil.
# Sem ele você teria que fazer tudo na mão. Com ele você ganha:


# O que ele faz?        

Serializers --->: (Converte objetos Python em JSON automaticamente)                    

APIView --->: (Classes prontas pra criar endpoints POST, GET)

Status codes --->: (status.HTTP_200_OK, HTTP_400_BAD_REQUEST)

Autenticação --->:   (Token, Session, JWT)

Browsable API --->:   (Interface visual pra testar seus endpoints no navegador)
