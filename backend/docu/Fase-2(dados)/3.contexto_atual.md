O que estamos construindo é um AGENTE que usa uma rede neural pronta.
Pensa assim:
Rede Neural (Gemini)  →  é o cérebro pronto
Nosso código          →  é como ensinamos esse cérebro
                          a trabalhar com nossos dados

Analogia mais clara:
Gemini      =  um médico formado e experiente
Nosso agente =  o consultório que esse médico trabalha
                com os instrumentos certos (datasets)
                com as regras certas (guardrails)
                com os casos certos (escopo)
Nós não criamos o médico — contratamos um médico pronto e montamos o consultório ideal pra ele trabalhar.

# O que cada parte do nosso código faz:
intent_service.py  →  interpreta o que o paciente quer
data_service.py    →  busca os exames (dados reais)
prompt_builder.py  →  prepara o briefing pro médico
gemini_service.py  →  o médico dá o diagnóstico
guardrails.py      →  as regras éticas do consultório

# O termo correto pro que estamos fazendo é:
Engenharia de Prompt  +  Integração de API  +  RAG simplificado

RAG (Retrieval Augmented Generation) = buscar dados reais antes de gerar a resposta. É exatamente o que fazemos com o pandas antes de chamar o Gemini. Isso inclusive é uma das técnicas mais usadas na indústria hoje em dia. ✅
