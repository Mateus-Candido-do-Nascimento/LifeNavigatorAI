config/urls.py     ← só tinha o admin
chat/urls.py       ← não existia
chat/views.py      ← existia mas não era acessível
services/          ← existiam mas ninguém orquestrava

React
  │
  │  POST /api/chat/
  ▼
config/urls.py         ← recebe qualquer /api/...
  │  inclui chat.urls
  ▼
chat/urls.py           ← direciona /api/chat/ pra ChatView
  │
  ▼
chat/views.py          ← orquestra tudo
  │
  ├─► guardrails       → mensagem segura?
  ├─► intent_service   → o que o usuário quer?
  ├─► data_service     → busca dados reais
  ├─► prompt_builder   → monta o prompt
  ├─► groq_service     → gera resposta
  └─► models           → salva no banco
  │
  ▼
React recebe o JSON com a resposta
