```
**Resumo visual:**
```
views.py importa
    │
    ├── DRF          → ferramentas de API
    ├── services/    → toda a lógica de negócio
    └── models.py    → acesso ao banco
O ponto importante é que a view não tem lógica — ela só importa e orquestra. Toda a inteligência está nos services. Isso é separação de responsabilidades na prática. ✅
