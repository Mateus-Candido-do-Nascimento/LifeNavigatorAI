---

**Por que Vite e não Create React App?**
```
Create React App  → antigo, lento, pesado
Vite              → moderno, rápido, leve
```

Vite é o padrão do mercado hoje. ✅

---

**O que o Vite cria automaticamente:**
```
frontend/
├── public/          → arquivos estáticos
├── src/
│   ├── App.jsx      → componente principal ← vamos mexer aqui
│   ├── main.jsx     → ponto de entrada da aplicação
│   └── index.css    → estilos globais
├── index.html       → HTML base
├── package.json     → dependências (igual requirements.txt do Python)
└── vite.config.js   → configurações do Vite


Agora entra na pasta e instala as dependências:
powershellcd frontend
npm install
npm install axios
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

# Antes de codar, vamos entender o main.jsx:
jsximport { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import './index.css'
import App from './App.jsx'

createRoot(document.getElementById('root')).render(
  <StrictMode>
    <App />
  </StrictMode>,
)
Esse arquivo é o ponto de entrada — ele pega o App.jsx e injeta dentro do <div id="root"> do index.html. Você nunca vai precisar mexer nele.
